Optical Character Recognition

Or commonly called OCR is a technology that recognizes text within an image. There are different ways of achieving OCR one common approach is to use deep learning. Deep learning need a lot of resources and powerful computer and GPU to rune them.

So I tried a different approach where real time OCR was the primary goal.

OCR with TensorFlow

You can use TensorFlow with the pre-trained OCR model keras-ocr. It consists of two models: one to localize the text in the image and the other to recognize the text.

Your images are processed by the server that performs OCR and sends you the predicted text. Your images are not stored or used for training the model; only the text is saved for 2 days to enable you to copy it. So, this approach is ideal for working with sensitive images.

The downside is that it is less precise and takes a bit longer to run. Currently, I am using a pre-trained model, but retraining the model for better results would be good, it's on my to-do list.

You can try it using the interface below. Be sure to select "OCR with TensorFlow" before submitting. I have chosen to enable auto-submit for faster responses, so users don't have to wait too long to see the results.

Link to keras-ocr documentation and GitHub repository.

API OCR

I use the OCR.space API to perform optical character recognition on images. It provides a good accuracy and is fast.

You can try it using the interface below. The API OCR is already preselected, give an image, and the text is yours.

I have enabled auto-submit for faster responses, so don't have to wait too long to see the results.

Please note that only optical character recognition in English is supported.
